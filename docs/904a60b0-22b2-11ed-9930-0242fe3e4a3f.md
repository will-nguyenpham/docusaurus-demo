# <a id="id" class="anchor_top_offset"/><a id="ariaid-title1" class="anchor_top_offset"/>How to perform Database Testing using <span xmlns="http://www.w3.org/1999/xhtml" className="ph">Katalon Studio</span>

With custom keywords, you can connect to databases and perform database testing. This tutorial describes details on how to create custom keywords for database testing in Katalon Studio.</p>
